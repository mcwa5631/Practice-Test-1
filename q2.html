Writing in Google Docs vs HTML

Writing in Google Docs and HTML is very different for three reasons. Firstly, how you format text. Secondly, because of how you must check your text output. Thirdly, the functions available with using each one.

Text Fromatting

One difference between writing in Google Docs Vs an html document is that in Google Docs you can automatically assign special chaarcters and format them however you want using the top toolbar. In html you must use code to format text such as "strong" to bold words.

Text Output

Another difference in html is that you must contsanly refresh your website in order to see how your text appears. In Google Docs you can just see the results of what you typed out.

Functions Available

Google Docs has many different functions available for users to type out whatever they need.
